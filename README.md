# NoReact
# 这个项目一个运行在nodejs和浏览器端的前端框架，让我们用更优雅的方式书写前后端分离的多页面项目。
## 开始开发
运行命令
```
npm run start
```
## 目录介绍
* sample  
组件的测试数据文件夹，存放所有组件的测试数据，注意文件名不能重复
* src  
包含运行在服务端和浏览器端的源代码
  * start.ts  
  nodejs启动文件
  * app.json  
  配置文件
  * www  
  运行在浏览器端的代码，包含所有的组件以及其他任何需要运行在浏览器端的代码
  * server  
  运行在nodejs中的代码
